# Dotfiles

Personal configuration files and scripts.

## Installation

```bash
git clone https://github.com/omarkohl/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

## What's Where

- [bin/](bin/) - Executable scripts and tools
- [shell/](shell/) - Shell configuration
- [git/](git/) - Git configuration
- [config/](config/) - Application configs (vim, nvim, etc.)

See individual directories and config files for specific documentation.
